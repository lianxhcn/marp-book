Guides and tips for making Marp slides.

- [在线阅读](https://lianxhcn.github.io/marp-book/)
- [PDF 版](https://lianxhcn.github.io/marp-book/Marp-book.pdf)

这本书是借助 [Claude code](https://claude.ai/share/9e3104dc-fbef-4540-90ae-62c0a6a98447) 生成的，耗时约 10 小时。虽然整体上不错，但难免错漏。欢迎大家提出修改意见和建议。

你可以点击本仓库上方的 **Fork** 按钮，将本仓库复制到你的 GitHub 账号下，然后在你的仓库中进行修改和编辑，最后通过 **Pull Request** 向我提交修改建议。

Fork 后，你也可以进一步使用 Github Desktop 或者 Git 命令行工具将仓库克隆到本地进行编辑和修改。`[template]` 目录下的 Markdown 文件是 Marp 幻灯片的模板，你可以参考这些文件来创建自己的幻灯片。
